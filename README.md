# LazyChilla AD Workshop

**Extension for [AutoDrive](https://www.farming-simulator.com) (FS25_AutoDrive) — adds a one-click Workshop button to the AutoDrive HUD.**

## Deutsch

### Was macht der Mod?
LazyChilla AD Workshop fügt dem AutoDrive-HUD einen zusätzlichen Button hinzu, mit dem ein Fahrzeug per Klick zu einem **globalen Werkstatt-Ziel** geschickt werden kann — ohne jedes Mal das Ziel-Dropdown durchsuchen zu müssen. Eine Hofwerkstatt, ein Klick, für alle Fahrzeuge.

### Features
- **Ein Klick zur Werkstatt**: Icon im AD-HUD anklicken, Fahrzeug fährt automatisch zur hinterlegten Werkstatt
- **Globales Ziel**: Einmal setzen, für alle Fahrzeuge auf dem Hof nutzbar (kein Ziel pro Fahrzeug nötig)
- **Rechtsklick zum Setzen**: Aktuell im AD-Dropdown gewähltes Ziel wird als Werkstatt gespeichert
- **Eigene Tastenbelegung**: Standard `STRG + ALT + W`, frei änderbar in den Spieleinstellungen
- **Kein Eingriff in AutoDrive**: Reine Erweiterung über Lua-Hooks, keine AD-Dateien werden verändert
- **Hover-Sichtbarkeit**: Icon erscheint nur wenn die Maus über dem AD-HUD ist, stört nicht beim Fahren

### Installation
1. [FS25_AutoDrive](https://github.com/Stephan-S/FS25_AutoDrive) muss installiert sein (Voraussetzung)
2. `FS25_AD_Extension.zip` in den `mods`-Ordner von Farming Simulator 25 kopieren
3. Spiel starten, Mod im Mod-Hub aktivieren

### Bedienung
| Aktion | Wie |
|---|---|
| Werkstatt-Ziel setzen | Im AD-HUD gewünschten Marker auswählen, dann **Rechtsklick** auf das Werkstatt-Icon |
| Zur Werkstatt fahren | **Linksklick** auf das Werkstatt-Icon, oder Taste `STRG+ALT+W` |
| Icon finden | Maus über das AD-HUD bewegen — Icon erscheint oberhalb der Titelleiste |

### Voraussetzungen
- Farming Simulator 25
- FS25_AutoDrive (aktuelle Version)

## English

### What does this mod do?
LazyChilla AD Workshop adds an extra button to the AutoDrive HUD that sends a vehicle to a **global workshop destination** with a single click — no need to scroll through the destination dropdown every time. One farm workshop, one click, works for every vehicle.

### Features
- **One click to the workshop**: click the icon in the AD HUD, the vehicle drives to the saved workshop automatically
- **Global destination**: set once, usable by every vehicle on the farm (no per-vehicle setup needed)
- **Right-click to set**: the currently selected AD dropdown destination is saved as the workshop
- **Custom keybinding**: default `CTRL + ALT + W`, freely remappable in game settings
- **No AutoDrive files modified**: pure extension via Lua hooks, AD itself stays untouched
- **Hover visibility**: the icon only appears while the mouse is over the AD HUD, stays out of the way while driving

### Installation
1. [FS25_AutoDrive](https://github.com/Stephan-S/FS25_AutoDrive) must be installed (required dependency)
2. Copy `FS25_AD_Extension.zip` into your Farming Simulator 25 `mods` folder
3. Launch the game, enable the mod in the mod hub

### Usage
| Action | How |
|---|---|
| Set workshop destination | Select the desired marker in the AD HUD, then **right-click** the workshop icon |
| Drive to workshop | **Left-click** the workshop icon, or press `CTRL+ALT+W` |
| Find the icon | Move the mouse over the AD HUD — the icon appears above the title bar |

### Requirements
- Farming Simulator 25
- FS25_AutoDrive (current version)

## Português

### O que este mod faz?
LazyChilla AD Workshop adiciona um botão extra ao HUD do AutoDrive que envia um veículo para um **destino de oficina global** com um único clique — sem precisar percorrer o menu de destinos toda vez. Uma oficina da fazenda, um clique, válido para todos os veículos.

### Funcionalidades
- **Um clique para a oficina**: clique no ícone no HUD do AD, o veículo segue automaticamente para a oficina salva
- **Destino global**: defina uma vez, utilizável por todos os veículos da fazenda
- **Clique direito para definir**: o destino atualmente selecionado no menu do AD é salvo como oficina
- **Tecla de atalho personalizável**: padrão `CTRL + ALT + W`, livremente alterável nas configurações do jogo
- **Nenhuma alteração no AutoDrive**: extensão pura via hooks Lua, os arquivos do AD permanecem intactos
- **Visibilidade ao passar o mouse**: o ícone só aparece quando o mouse está sobre o HUD do AD

### Instalação
1. [FS25_AutoDrive](https://github.com/Stephan-S/FS25_AutoDrive) precisa estar instalado (dependência obrigatória)
2. Copie `FS25_AD_Extension.zip` para a pasta `mods` do Farming Simulator 25
3. Inicie o jogo e ative o mod no mod hub

### Uso
| Ação | Como |
|---|---|
| Definir destino da oficina | Selecione o marcador desejado no HUD do AD, depois **clique direito** no ícone da oficina |
| Ir para a oficina | **Clique esquerdo** no ícone da oficina, ou tecla `CTRL+ALT+W` |
| Encontrar o ícone | Mova o mouse sobre o HUD do AD — o ícone aparece acima da barra de título |
